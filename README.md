Notre application Android permettant de consulter des articles de presse récents via une API de news newsapi.org.  

## Technologies utilisées

- Kotlin  
- Jetpack Compose  
- Architecture MVVM  
- Retrofit (appels API)  
- Room (stockage local)  
- Dagger Hilt (injection de dépendances)  
- Coroutines & Flow  
- Paging  
- WorkManager  
- Coil (chargement d’images)

## Fonctionnalités

- Affichage des dernières actualités  
- Recherche d’articles  
- Filtrage par source  
- Lecture complète des articles via WebView  
- Sauvegarde d’articles en favoris  
- Suppression d’articles des favoris    
- Partager des articles
